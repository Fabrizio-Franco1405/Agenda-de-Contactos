# Agenda de Contactos
# 
# Este repositorio contiene una aplicación de consola desarrollada en C#. La aplicación sirve como una agenda de contacto, permitiendo a los usuarios almacenar, recuperar, actualizar y eliminar información de contacto. La aplicación es simple pero eficiente, y demuestra el uso de varias características clave de C#, incluyendo la manipulación de datos, el control de flujo y la programación orientada a objetos. Este proyecto es ideal para aquellos que buscan entender los fundamentos de C# y cómo se puede utilizar para crear aplicaciones prácticas y útiles.
